# Portfolio Site

This is a Django-based portfolio website designed to showcase software engineering projects, skills, and professional information. It features dynamic content management through Django admin, project display with images and details, a blog section for articles, and a tags feature for organizing content.

## Features

- **Projects Display**: Showcases software engineering projects with images, descriptions, tags, and links to live versions or source code repositories.
- **Blog**: A section for writing and displaying blog posts, with support for tags and comments.
- **Skills**: Lists technical skills and proficiency levels.
- **About Me**: A personal introduction and professional background.
- **Contact Information**: Provides ways to contact or connect on professional networks.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Python (3.8 or higher)
- Django (5.0.3)
