# Portfolio Site

This is a Django/React portfolio and blog website designed to showcase software engineering projects, skills, and professional information. It features dynamic content management through Django admin, project display with images and details, a blog section for articles, and a tags feature for organizing content. If you'd like to reach out to me, you are welcome to do that too.

## Features

- **Projects Display**: Showcases software engineering projects with images, descriptions, tags, and links to source code repositories.
- **Blog**: A section for writing and displaying blog posts, with support for tags and comments.
- **Skills**: Lists technical skills and proficiency levels.
- **About Me**: A personal introduction and professional background.
- **Contact Information**: Provides ways to contact or connect on professional networks.
